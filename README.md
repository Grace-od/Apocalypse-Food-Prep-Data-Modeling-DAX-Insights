# Apocalypse Food Prep: Power BI Data Modeling & DAX Analysis

## 📌 Overview
This repository contains a completed Power BI project focused on advanced **Data Modeling** and **DAX (Data Analysis Expressions)**. Using the "Apocalypse Food Prep" datasets, I transformed raw data into a relational model to extract meaningful insights regarding food supply and preparation trends.

## 🚀 Key Tasks Completed

### 1. Data Modeling & Relationships
* **Schema Design:** Structured the data into a Star Schema for efficient analytical performance.
* **Relationship Management:** Established one-to-many relationships between fact and dimension tables.
* **Cardinality:** Configured relationship directions to ensure accurate data filtering and integrity.

### 2. DAX Calculations
* **Calculated Columns:** Built row-level logic to categorize data and enhance filtering capabilities.
* **Measures:** Authored dynamic DAX measures for real-time aggregations and KPI tracking.
* **Functional Logic:** Applied functions to derive specific insights from the `DAX Tutorial` dataset.

## 🛠️ Technical Stack
* **Tool:** Power BI Desktop
* **Language:** DAX (Data Analysis Expressions)
* **Data Source:** Microsoft Excel (`Relationship Tutorial.xlsx`, `DAX Tutorial.xlsx`)

## 📖 How to Use
1. Clone this repository to your local machine.
2. Ensure you have the latest version of [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed.
3. Open the `.pbix` file to view the data model and interactive insights.
